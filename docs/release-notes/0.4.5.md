- Areas of Focus for projects, plus project tags, are now supported across core and UI.
- Weekly recurrence supports specific weekdays (e.g., Mon/Wed/Fri).
- Android share intent support so Mindwtr can appear in the system Share menu.
- Settings are simplified and reorganized to reduce cognitive load.
